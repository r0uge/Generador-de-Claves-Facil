# Generador de Claves de Facil Lectura

>Autor: Agustin Alvarez
>Fecha Compilación: 22/5/2024
>Versión: 3.0
>Lenguaje: PowerShell

##Descripción

Permite la generación de claves que es de facil lectura conformada por tres palabras aleatoreas unidas por un separador y con una combinación númerica aleatoria de tres dígitos. Ademas adiciona una mayuscula en una palabra en forma aleatoria.
Máximo 14 caracteres genera.
El diccionario cuenta con 46 palabras.
El binario se encuentra compilado para plataformas Windows, mediante la herramienta Win-PS2EXE
Requiere que el sistema operativo cuente con PowerShell para su ejecución
